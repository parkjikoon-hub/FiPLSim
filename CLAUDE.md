# 소화배관 시뮬레이터 프로젝트 - Claude 지침서

## 소통 규칙

사용자는 비개발자이므로 다음을 항상 준수합니다:

1. **매 단계마다 설명하기**: 무엇을 하고 있는지, 왜 하는지를 쉬운 말로 설명합니다.
   - 예: "지금 필요한 프로그램들을 설치하고 있어요. 이건 앱을 실행하는 데 필요한 부품들이에요."
2. **전문 용어 피하기**: 기술 용어를 사용할 때는 반드시 쉬운 설명을 함께 제공합니다.
   - 예: "pip (파이썬 패키지 설치 도구)"
3. **진행 상황 표시**: 이모지와 단계 번호를 사용하여 현재 진행 상황을 시각적으로 표시합니다.
   - 예: "✅ 1단계 완료 → 🔄 2단계 진행 중..."
4. **에러 발생 시**: 에러가 무엇인지, 왜 발생했는지, 어떻게 해결할 건지를 쉽게 설명합니다.
5. **사용자 행동 필요 시**: 사용자가 직접 해야 할 일이 있으면 명확하게 번호를 매겨 안내합니다.

## 프로젝트 정보

- **실행 방법**: `streamlit run app.py`
- **Python**: `python3` (버전 3.14)
- **주요 파일**: constants.py, hydraulics.py, pipe_network.py, pump.py, simulation.py, app.py
