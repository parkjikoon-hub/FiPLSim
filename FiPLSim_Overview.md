# FiPLSim (Fire Protection Pipe Line Simulator) — 프로그램 개요

> 최종 업데이트: 2026-03-10
> 버전: v2.0 (NFTC 103 반영, 비균일 비드 모델 포함)

---

## 1. 프로그램 목적

FiPLSim은 **소화배관 시스템의 수리계산 및 통계 분석** 시뮬레이터입니다.

**핵심 비교 대상:**
- **Case A (기존 기술)**: 용접 이음쇠에 내면 비드(돌출)가 존재 → 추가 압력 손실 발생
- **Case B (형상제어 신기술)**: 비드를 제거/제어 → 기본 손실만 발생

**핵심 질문:**
> "용접 비드가 소화배관의 말단 방수압에 얼마나 영향을 미치는가?"
> "어떤 조건에서 비드로 인해 법적 기준(0.1 MPa)을 만족하지 못하는가?"

## 2. 이론적 근거 및 수식

### 2.1 주손실 — Darcy-Weisbach 방정식

배관 내 유체가 직관을 흐를 때 발생하는 마찰 손실:

```
h_f = f × (L / D) × (V² / 2g)   [m]
ΔP = ρ × g × h_f / 10⁶          [MPa]
```

| 기호 | 의미 | 단위 |
|------|------|------|
| f | Darcy 마찰계수 (무차원) | - |
| L | 배관 길이 | m |
| D | 배관 내경 | m |
| V | 유속 | m/s |
| g | 중력가속도 (9.81) | m/s² |
| ρ | 유체 밀도 | kg/m³ |

### 2.2 마찰계수 — Colebrook-White 방정식

난류(Re ≥ 2300) 조건에서 마찰계수를 구하는 반복 수렴식:

```
1/√f = -2.0 × log₁₀[ (ε/D)/3.7 + 2.51/(Re × √f) ]
```

- **층류** (Re < 2300): f = 64 / Re
- **Re** = V × D / ν (레이놀즈 수)
- **ε** = 0.045mm (탄소강 절대조도, 기본값)
- 초기 추정: Swamee-Jain 근사식, 최대 10회 반복 수렴

### 2.3 부차손실 (국부 손실)

배관 이음쇠, 밸브, 분기점 등에서 발생하는 국부적 손실:

```
h_m = K × (V² / 2g)   [m]
```

K값은 부속류 종류에 따라 결정:

| 부속류 | K값 | 비고 |
|--------|-----|------|
| K1 (배관이음쇠) | 0.5 (기본) | 비드 0mm 기준 |
| K2 (헤드이음쇠) | 2.5 | 스프링클러 헤드 분기 |
| K3 (분기 입구) | 1.0 | Tee-Branch 분기 |
| K_TEE_RUN | 0.3 | 교차배관 직진 |
| 알람밸브 | 2.0 | 습식 |
| 체크밸브 | 2.0 | 스윙형 |
| 게이트밸브 | 0.15 | 전개 |

### 2.4 비드 손실 모델 — K_eff 수식

**핵심 수식:**
```
K_eff = K_base × (D / D_eff)⁴
D_eff = D - 2 × h_b
```

| 기호 | 의미 |
|------|------|
| K_base | 비드 없을 때의 기본 K값 (0.5) |
| D | 원래 배관 내경 (mm) |
| D_eff | 비드로 축소된 유효 내경 (mm) |
| h_b | 비드 높이 (mm) |

**이론적 근거:**
- 연속 방정식: Q = A × V → V ∝ 1/D² (내경 축소 시 유속 증가)
- 손실 ∝ V² → 손실 ∝ 1/D⁴
- **Borda-Carnot 급축소 이론**과 일치 (면적비의 제곱 = 직경비의 4제곱)
- **Idelchik** "Handbook of Hydraulic Resistance" 내부 돌출물 손실과 일치
- K_base = 0.5는 **CRANE TP-410** 용접이음쇠 표준값

**비균일 모델 (v2.0 추가):**
- 균일 모델: 모든 비드 동일 높이 (예: 전부 1.5mm)
- 비균일 모델: 각 접합부마다 정규분포 N(μ, σ²)로 높이 변동
- 4제곱 비선형 수식으로 인해 비균일 시 평균 손실 증가 (젠센 부등식)

### 2.5 Hardy-Cross 반복법 (Full Grid 배관망)

교차배관 2개(TOP/BOT)가 평행 배치된 격자형 배관망에서 루프별 유량 균형 수렴:

```
보정 유량: ΔQ = -Σ(h_i) / Σ(2|h_i|/Q_i)
수렴 조건: 각 루프의 수두 불균형 < 0.001 m
이완 계수: 0.5 (기본), 0.1~1.0 조절 가능
```

## 7. 프로그램 구조

### 7.1 파일 구성

```
FiPLSim/
├── constants.py      물리 상수, 배관 치수, K값, 기본 파라미터
├── hydraulics.py     유체역학 계산 (Darcy, Colebrook, K_eff)
├── pipe_network.py   동적 배관망 생성 및 압력 계산
├── hardy_cross.py    Full Grid 배관망 (Hardy-Cross 솔버)
├── simulation.py     MC 시뮬레이션, 민감도, 변수 스캔
├── pump.py           펌프 P-Q 곡선, 운전점, 에너지 절감
├── app.py            Streamlit UI (8탭 대시보드)
├── test_*.py         테스트 4개 (195개 테스트 항목)
└── FiPLSim_Overview.md  ← 이 파일
```

### 7.2 데이터 흐름

```
사용자 입력 (사이드바)
  │
  ├─ 배관망 크기 (n가지 × m헤드)
  ├─ 운전 조건 (압력, 유량)
  ├─ 비드 설정 (높이, 개수, 변동)
  └─ 시뮬레이션 파라미터
  │
  ▼
compare_dynamic_cases_with_topology()
  ├─ generate_dynamic_system()  → DynamicSystem 객체
  │    ├─ 교차배관 자동 구경 선정
  │    ├─ 가지배관 구경 자동 선정
  │    ├─ 이음쇠 비드 K값 계산
  │    └─ 직관 용접 비드 생성/배치
  │
  ├─ calculate_dynamic_system()  → 압력 프로파일
  │    ├─ Step 0: 밸브/기기류 손실
  │    ├─ Step 1: 교차배관 손실 누적
  │    ├─ Step 2: 각 가지배관 압력 순차 계산
  │    └─ Step 3: 3항 분리 손실 집계
  │
  └─ 결과: 최악 말단 압력, 개선율, PASS/FAIL
  │
  ▼
UI 표시 (8개 탭)
```

### 7.3 압력 계산 경로 (최악 가지배관)

```
입구 압력 P_inlet
  │
  ├─ [-] 밸브/기기류 손실 (라이저)
  │
  ├─ [-] 교차배관 직관 마찰 × (n-1)구간
  │     각 구간: f × (L/D) × (V²/2g)
  │
  ├─ [-] Tee-Run 손실 × (n-1)개
  │     각: K_TEE_RUN × (V²/2g)
  │
  ├─ [-] 분기 입구 손실 K3
  │     K3 × (V_branch²/2g)
  │
  ├─ [-] 입구관 마찰 (0.3m 65A 등)
  │
  └─ 가지배관 순환 (m개 헤드)
       │
       for h = 0 to m-1:
         segment_flow = branch_flow - h × head_flow
         │
         ├─ [-] 직관 마찰: f × (head_spacing/D) × (V²/2g)
         ├─ [-] 이음쇠 비드 K1: K_eff × (V²/2g)
         ├─ [-] 헤드 K2: K2 × (V²/2g)
         └─ [-] 직관 비드들: Σ[K_bead × (V²/2g)]
       │
       ▼
  말단 압력 P_terminal  ← 이 값이 0.1 MPa 이상이어야 PASS
```

## 8. 논문 작성 참고사항

### 8.1 시뮬레이터 명칭 및 인용

- **프로그램명**: FiPLSim (Fire Protection Pipe Line Simulator)
- **개발 환경**: Python 3.14, Streamlit, NumPy, SciPy
- **핵심 알고리즘**: Darcy-Weisbach + Colebrook-White (반복법) + K-factor 방법론
- **비드 모델**: K = K_base × (D/D_eff)⁴ (Borda-Carnot 기반, CRANE TP-410 준용)

### 8.2 참고 문헌 목록

| 이론/수식 | 출처 |
|-----------|------|
| Darcy-Weisbach 주손실 | 기초 유체역학 표준 |
| Colebrook-White 마찰계수 | Colebrook (1939) |
| K-factor 방법론 | CRANE TP-410 |
| 내부 돌출물 손실 | Idelchik, "Handbook of Hydraulic Resistance" |
| 비드 K = K_base×(D/D_eff)⁴ | Borda-Carnot 급축소 이론 유도 |
| 관경 선정 기준 | NFTC 103 (기술기준) 표 2.5.3.3 |
| 방수압/방수량 기준 | NFPC 103 (성능기준) |
| Hardy-Cross | Cross, H. (1936) |

### 8.3 수식 표기 시 주의사항

- **D_eff**: 프로그램에서는 `D - 2×h_b` (양측 돌출 가정, 환상 비드)
- **K_base = 0.5**: 논문에서 "비드가 없는 표준 용접이음쇠의 K값"으로 명시
- **(D/D_eff)⁴**: "유량 보존 조건에서 단면적 축소에 따른 유속 증가의 제곱 효과"로 설명
- **Pf**: P(P_terminal < 0.1 MPa) — "법적 기준 미달 확률"

### 8.6 논문에서 언급해야 할 모델 한계점 (Limitations)

1. **환상 비드 가정**: 실제 비드는 부분적이고 불규칙하나, 프로그램은 완전 환형으로 가정 (보수적 추정)
2. **비균일 비드**: v2.0에서 정규분포 변동 모델 추가, 젠센 부등식 효과 반영
3. **단일 배관 구성**: 현재 8헤드 고정 구성. 다른 구성은 파라미터 변경으로 시뮬레이션 가능
4. **K_base 고정**: Re > 10,000 이상에서 K가 Re에 거의 무관하므로 0.5 고정 (±10% 민감도 분석 가능)
5. **온도 영향**: 20°C 기준 물성치 사용. 온도 변화에 따른 점성 변화 미반영
6. **필드 검증**: 실험실/현장 실측 데이터와의 교차 검증은 향후 과제
7. **균등 유량 가정**: 각 헤드에 동일 유량 배분 (Tree 모드). Grid 모드에서는 Hardy-Cross로 유량 균형
